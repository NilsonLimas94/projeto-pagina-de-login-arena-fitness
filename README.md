# Arena Fitness -- Interface de Login

Projeto de **interface de login institucional** desenvolvido para a
academia **Arena Fitness**, com foco em usabilidade, clareza visual e
identidade de marca.
O objetivo é oferecer uma experiência simples e moderna para
autenticação de usuários em plataformas web.

------------------------------------------------------------------------

## Objetivo do Projeto

Criar uma página de login responsiva e visualmente agradável que possa
ser utilizada como base para: Sistemas internos da academia, portais
de alunos, aplicações web com autenticação de usuários.

------------------------------------------------------------------------

## Visão Geral da Interface

A página de login contém:

-   Mensagem de boas-vindas ao usuário
-   Formulário de autenticação com:
    -   Campo de e-mail
    -   Campo de senha
-   Link para recuperação de senha
-   Opção de login via Google
-   Link para criação de nova conta
-   Identidade visual com logotipo da Arena Fitness

------------------------------------------------------------------------

## Tecnologias Utilizadas

-   **HTML5**
    -   Estrutura semântica e organizada
-   **CSS3**
    -   Estilização externa

    -   Compatível com dispositivos móveis e desktops

## Como Executar o Projeto

1.  Clone o repositório:

    ``` bash
    git clone https://github.com/seu-usuario/seu-repositorio.git
    ```

2.  Acesse a pasta do projeto:

    ``` bash
    cd arena-fitness-login
    ```

3.  Abra o arquivo `index.html` em qualquer navegador moderno.

------------------------------------------------------------------------

## Considerações Técnicas

-   Este projeto é **exclusivamente front-end**
-   Não possui integração com banco de dados ou backend
-   Os formulários não realizam autenticação real
-   Ideal para estudos, prototipação ou base para projetos maiores
